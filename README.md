## 🔐 Social Login Configuration

### Google Sign-In

Google Sign-In is the simplest to configure as Firebase handles most of the setup automatically.

#### Step 1: Enable Google Sign-In in Firebase Console

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Select your project **"smockly"**
3. In the left sidebar, click **Authentication** → **Sign-in method** tab
4. Find **"Google"** in the providers list and click on it

#### Step 2: Configure Google Provider

In the dialog that appears:

1. Toggle **"Enable"** to ON
2. Add your **Project support email** (select from dropdown)
3. Click **"Save"**

> ✅ **That's it!** Google Sign-In works with the default configuration — no Client ID or Client Secret needed because Firebase automatically handles this for you.
